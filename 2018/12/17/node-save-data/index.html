  • Node中常用的存储数据的方式
    • 无服务器的数据存储
      1. 内存存储: 内存存储的理想用途是存放少量经常使用的数据;其读写速度快，但是在服务器重启之后数据便会丢失.
      2. 基于文件的存储: 用文件系统存放数据,可以使用它来做数据持久化，数据在程序和服务器重启之后依然存在.
    • 关系型数据库
      关系型数据库可以存储复杂的信息，并且查询起来比较容易
      1. mySql
      2. PostgreSQl
